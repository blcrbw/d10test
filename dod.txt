1) Посложнее

Необходимо сделать сайт на базе CMS Drupal версии 10, на котором будет производиться поиск и выдача товара. У товара
есть бренд и до 5-ти разных характеристик (названия характеристик значения не имеют), однако товары могут иметь разные
изображения (изображение не является характеристикой). Для примера можно использовать пять одинаковых картинок на все
товары.

Необходимо:
1) "Развернуть" сайт на базе темы Bootstrap 5;
2) Реализовать импорт товаров из файла формата CSV;
3) Реализовать минимальную функциональность для вывода товаров на общей странице каталога. При выборе бренда переходить
на страницу бренда.
4) Реализовать страницу для вывода товаров на странице бренда;
5) Реализовать страницу экспорта товаров в формате CSV;
6) Реализовать страницу для вывода товаров на странице бренда с динамическим URL. Что это значит: есть URL вида
http://site.com/catalog/BrandName/filter_name/filter_value или http://site.com/catalog/BrandName/filter_name-filter_value,
либо любой другой удобный для Вас формат динамических фильтров (выбрано значений может быть несколько, фильтров тоже
несколько), http://site.com/catalog/BrandName - это URL страницы бренда, filter_name/filter_value – фильтры, которые
выбрал пользователь по которым необходимо показать товар.

Для реализации пункта 6 может понадобиться написать модуль.

2) Попроще

Нужно разработать модуль импорта данных из файла XLS в Drupal 10.

Файл содержит 5 вкладок. Импорт должен осуществляться из третьей.
